Terms of service and security notice.

By using this dashboard you agree to the site's security, session, and operational controls. These terms explain what happens if repeated failed login attempts occur.

What the site stores

GigOptimizer stores session cookies for authenticated dashboard use, marketplace settings you save, and admin-approved operational data such as reports and queue actions.

Failed login protection

After 3 failed login attempts from the same browser or device, the site may request a consent-based camera snapshot. If granted, that snapshot is shown only inside the authenticated admin dashboard for review.

What device details are recorded

The site records limited security metadata such as IP address, browser user agent, language, screen size, timezone, and touch support when available. It does not attempt to silently collect hidden phone identifiers.

Admin review

Security captures remain visible only to the admin account. The admin can save or discard each capture from the dashboard.

Consent choices

If you decline cookies or camera consent, the site will still allow normal login attempts, but it will skip the camera capture step and record that consent was declined.
