Install Fiverr Capture Extension

Use this extension to capture the Fiverr page you are already viewing in Chrome and send the visible cards into GigOptimizer Pro for live comparison.

Download ZIP Back to dashboard

Connection details

API base URL {{ install.api_base_url }}
Extension API token {{ install.api_token or 'Not configured' }}

Steps

  1. Open chrome://extensions.
  2. Turn on Developer mode.
  3. Click Load unpacked if you extracted the ZIP, or unzip the downloaded extension first.
  4. Select the fiverr-market-capture folder.
  5. Open the extension popup and paste the API base URL and token from above.
  6. Open a Fiverr search or gig page and click Capture current Fiverr page.

Expected behavior

  • The extension reads the visible Fiverr cards in your browser only.
  • It sends those visible cards to your private GigOptimizer backend.
  • Your dashboard compares them with your active Fiverr gig and updates the market recommendations.
  • The dashboard install banner disappears automatically once the extension is detected on animha.co.in.
